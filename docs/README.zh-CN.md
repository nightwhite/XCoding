# XCoding

面向 Vibe Coding 的新一代 IDE。

为 Vibe Coding 而生。轻量、简洁、高效。专注把“写代码”这件事做得更顺手：多项目、多任务、少打断、低占用。

[![XCoding 截图](images/screenshot.png)](images/screenshot.png)

Language: **简体中文** | [English](../README.md)

## 你会用它做什么

- 同时开多个项目，并排推进
- 一边写代码，一边跑命令，一边看页面
- 随时把问题交给 AI，再把改动一键应用或回滚

## 它有什么不一样

- 多项目：一个窗口里管理多个项目，`Cmd/Ctrl+1..8` 快速切换
- 多任务：编辑器、终端、预览、AI 面板在同一个工作台里
- 更轻：启动快、占用低；忙的时候也不“卡住你”
- 更省心：不走插件生态路线，内置的能力就是核心路径

## 本地开发

安装依赖：

`pnpm install`

启动：

`pnpm run dev`

常用命令：

- `pnpm run build`
- `pnpm run lint`
- `pnpm run typecheck`
